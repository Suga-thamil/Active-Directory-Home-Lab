# Security Event Analysis

## Objective

Analyze Windows Security Event Logs generated by Active Directory authentication activity.

---

## Environment

| Component | Value |
|------------|---------|
| Domain | corp.local |
| Domain Controller | DC01 |
| Client Workstation | CLIENT01 |

---

## Event Viewer Location

```text
Event Viewer
 └─ Windows Logs
     └─ Security
```

The Security log was monitored after enabling logon auditing through Group Policy.

---

## Event ID 4624 - Successful Logon

### Description

Event ID 4624 indicates that a user successfully authenticated to the system.

### Example Observed

| Field | Value |
|---------|---------|
| Event ID | 4624 |
| TargetUserName | alice |
| TargetDomainName | CORP.LOCAL |
| Authentication Package | Kerberos |
| Logon Type | 3 |

### Analysis

The event confirms:

- Successful authentication
- User account validation
- Kerberos authentication usage
- Active Directory communication functioning correctly

---

## Event ID 4625 - Failed Logon

### Description

Event ID 4625 indicates that an authentication attempt failed.

### Example Observed

| Field | Value |
|---------|---------|
| Event ID | 4625 |
| Authentication Failure | Yes |
| Logon Attempt | Unsuccessful |

### Analysis

Failed logon events can indicate:

- Incorrect passwords
- Invalid usernames
- Account lockouts
- Potential brute-force attempts

Security analysts frequently monitor Event ID 4625 to detect suspicious authentication activity.

---

## Security Significance

Authentication logs are valuable because they help identify:

- Unauthorized access attempts
- Password guessing attacks
- User login activity
- Potential insider threats
- Account misuse

Monitoring Event IDs 4624 and 4625 is a fundamental security monitoring practice.

---

## Lessons Learned

- Event Viewer provides detailed authentication records.
- Successful and failed logons generate distinct event IDs.
- Kerberos authentication events can be analyzed through Security Logs.
- Security auditing improves visibility into user activity.
- Event logs are important for incident investigation and threat detection.
